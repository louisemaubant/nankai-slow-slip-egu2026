## Abstract

We present a new method to detect and quantify short-term slow slip events (S-SSEs) 
along the Nankai subduction zone using LFE-guided decomposition of GNSS time series. 
Slip is inverted from LFE-associated displacements over 20 years (2004–2024), 
revealing systematic spatial and temporal variations in S-SSE activity relative 
to the locked megathrust.
